Privacy Policy

UdharBook — Aapka data, aapka control

📋 Overview

UdharBook ek personal finance app hai jo aapko apna udhar (loans given/received) track karne mein help karta hai. Hum aapki privacy ko seriously lete hain.

Yeh policy explain karti hai ki hum kya data collect karte hain, kyun karte hain, aur kaise use karte hain.

💡 Short version: Hum aapka data sirf app functionality ke liye use karte hain. Kisi third party ko sell nahi karte. Kabhi nahi.

📦 Kya Data Collect Karte Hain

Jo aap dete hain:

  • Contact names aur phone numbers (jo aap manually add karte ho)
  • Transaction amounts, notes, aur dates
  • Account information (email ya phone number for Google/Phone login)
  • App preferences (language, theme)

Jo automatically collect hota hai:

  • Device type aur OS version (crash reporting ke liye)
  • App usage patterns (anonymous, koi personal info nahi)

Jo hum collect nahi karte:

  • Aapke device contacts (hum permission nahi maangte)
  • Location data
  • Camera ya microphone access
  • Biometric data

☁️ Cloud Storage (Supabase)

Agar aap Google login ya Phone login use karte hain, aapka data securely Supabase (PostgreSQL database) mein store hota hai.

  • Data encrypted hai in transit (HTTPS/TLS)
  • Data encrypted hai at rest
  • Servers EU/US mein hain (Supabase infrastructure)
  • Aap kabhi bhi apna data delete kar sakte hain

Guest mode use karne par, aapka data sirf aapke device pe store hota hai — koi cloud sync nahi.

🔐 Authentication

Hum Google Sign-In aur Phone OTP authentication use karte hain. Yeh Supabase Auth ke through handle hota hai.

  • Google Sign-In se sirf aapka email aur naam liya jaata hai
  • Password hum kabhi store nahi karte
  • Phone OTP — sirf verification ke liye, stored nahi hota

🔔 Notifications

Hum local push notifications bhejte hain sirf aapki permission ke baad:

  • Aapke set kiye reminders (contact-wise)
  • EMI due date reminders
  • Daily balance summary (optional)

Hum koi marketing notifications ya promotional messages nahi bhejte.

💳 Subscriptions (Pro)

UdharBook Pro subscription Google Play Billing ke through process hoti hai.

  • Payment information directly Google handle karta hai — hum koi card details store nahi karte
  • Subscription status RevenueCat ke through verify hota hai
  • Kabhi bhi Google Play se subscription cancel kar sakte hain

👨‍👩‍👧 Third Party Services

Hum inhe use karte hain:

  • Supabase — database aur authentication (supabase.com/privacy)
  • Google Sign-In — authentication (policies.google.com/privacy)
  • RevenueCat — subscription management (revenuecat.com/privacy)
  • Expo (EAS) — app infrastructure (expo.dev/privacy)

Koi ads network, analytics platform, ya data broker use nahi kiya jaata.

🗑️ Data Deletion

Aap apna data kabhi bhi delete kar sakte hain:

  • Individual data: App ke andar contact ya transaction delete karo
  • Full account deletion: Settings → Logout karo — guest data automatically delete hota hai
  • Cloud data deletion: Humse contact karo, 30 din ke andar delete karenge

👶 Children's Privacy

UdharBook 18 saal se kam umra ke users ke liye nahi hai. Hum jaante bujhte 18 se kam ke users ka data collect nahi karte.

🔄 Policy Changes

Agar hum yeh policy update karte hain, aapko app ke through notify karenge. Continued use of the app after changes means you accept the updated policy.
