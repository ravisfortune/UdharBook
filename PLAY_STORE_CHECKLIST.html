Phase 4 — Subscription Setup
Google Play billing + RevenueCat integration
⚠️ Important: Play Store subscriptions ke liye Razorpay nahi chahiye. Google apna billing system use karta hai. Revenue ka 15–30% Google rakhta hai (pehle saal 15%, baad mein 30%).
✓
Play Console → Monetize → Subscriptions
Pehle ek app upload hona chahiye — subscriptions tab tab activate hota hai
✓
Monthly subscription create karo
Product ID: udharbook_pro_monthly · Price: ₹99/month · Grace period: 3 days
✓
Yearly subscription create karo
Product ID: udharbook_pro_yearly · Price: ₹799/year · 7-day free trial (optional)
✓
RevenueCat account banao Free
app.revenuecat.com — $2500/month revenue tak bilkul free · Google Play billing handle karta hai
✓
RevenueCat SDK install karo Code
npx expo install react-native-purchases · useProStore mein wire karo
✓
RevenueCat ko Play Console se connect karo
RevenueCat Dashboard → Google Play → Service Account credentials add karo
💡 RevenueCat kyun? Google Play billing API directly use karna complex hai. RevenueCat webhook, receipt validation, subscription status sab handle karta hai — tumhe sirf purchase() call karna hai.
